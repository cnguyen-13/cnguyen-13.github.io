Cookbook Reference

Project #2

Users looking for new and delicious recipes that meets their nutritional needs can do just that! Using the Spoonacular API, this project allows users to filter results based on ingredients, nutritional values, and much more. Recipes can be "liked" and saved in order to build a recipe list of interested recipes using local storage.

HTML, CSS/SCSS (BEM Convention), JavaScript, React (Custom Hooks, GSAP, React Router DOM), API consumption, Local Storage.
